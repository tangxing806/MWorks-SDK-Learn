> MWorks SDk安装后，在安装目录下存在以下目录:
>
> bin : 里面包含x86和x64的dll、lib、求解器和配置环境
> examples ：基于MWorks SDK 开发的示例列表
> include : MWorks SDK 依赖的第三方库头文件
> interface : MWorks SDK 头文件
> docs ：MWorks SDK 帮助文档
